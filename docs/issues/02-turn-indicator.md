## What to build

The sidebar displays a colored circle and text label ("White's Turn" / "Black's Turn") indicating whose turn it is. The board also shows a subtle glow or border highlight on the active player's edge (bottom for white, top for black in default orientation). Both update after every move, undo, and redo.

## Acceptance criteria

- [ ] Sidebar shows a white-filled circle and "White's Turn" at the start of the game
- [ ] Sidebar shows a black-filled circle and "Black's Turn" after white makes a move
- [ ] Board displays a glow or border accent on the bottom edge when it is white's turn
- [ ] Board displays a glow or border accent on the top edge when it is black's turn
- [ ] Indicator updates correctly after undo and redo

## Blocked by

None — can start immediately.
